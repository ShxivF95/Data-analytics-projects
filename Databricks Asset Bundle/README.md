Overview

This repository uses Databricks Asset Bundles (DAB) to manage and deploy Databricks resources using Infrastructure as Code principles.
It is designed for CI/CD automation with GitHub Actions and supports multi-environment deployments.

What is a Databricks Asset Bundle?

Databricks Asset Bundles provide a declarative way to define and deploy Databricks resources such as:

Jobs

Notebooks

Delta Live Tables (DLT) pipelines

Clusters

Permissions

Bundles are deployed using Databricks CLI v2, enabling repeatable, non-interactive deployments across environments.
